# 遊戲王中文卡表工具

一個純前端、開箱即用的 **遊戲王（Yu-Gi-Oh! OCG/TCG）中文卡片工具**。
提供中文卡查、現行環境流行構築、以及依你的條件自動生成卡組，並附卡圖方便辨識。

> ⚠️ 卡片資料為即時查詢，卡圖亦即時載入，因此**使用時需要能連上網路**。

## 功能

### 🔍 中文卡片搜索
- 直接以**中文或日文卡名**、效果關鍵字搜尋（例：`灰流丽`、`篝火`、`增殖`、`三战之才`）。
- 依卡片種類（怪獸／魔法／陷阱）快速篩選。
- 點任一卡片可看**完整中文效果**、攻守、屬性與日/英文名。
- 每張卡右上角「＋」可直接加入「我的卡組」。

### 🏆 現環境流行構築
- 收錄現行環境常見主題（雷火沸動機 Ryzeal、蛇眼、天盃龍、于貝爾、白森林、炎王…），
  附**中文戰術說明**、**構築骨架比例**與**關鍵卡卡圖**。
- 每個主題可「**載入系列全卡**」——即時從資料庫抓取該系列**所有卡片**（永遠是最新收錄）。
- 一鍵「**匯入到我的卡組**」，把整份範例構築帶進卡組編輯器。

> 流行構築為代表性**骨架範例**（依 2025 下半 ~ 2026 上半環境知識整理），比例僅供參考；
> 實際比賽請務必**對照最新禁限表**。

### 🛠️ 卡組生成器（依條件構築）
選擇你的條件，一鍵生成卡組骨架：
- **主題引擎**：雷火沸動機 / 蛇眼 / 天盃龍 / 于貝爾 / 白森林 / 炎王 / 純手坑 Goodstuff。
- **卡組風格**：連招 / 控制 / 快攻 OTK。
- **預算**：不限 / 中等 / 省錢（會依預算調整泛用手坑取捨）。
- **主卡組張數、手坑數量、破壞卡數量**：以滑桿自由調整。

生成結果可再「匯入到我的卡組」微調。

### 🗂️ 我的卡組
- 跨分頁共用的工作卡組，資料存於瀏覽器本機（localStorage），關掉頁面也不會消失。
- 自動檢查合法性（主卡組 40–60、額外/副 ≤15、每種卡 ≤3）。
- 一鍵**匯出 `.ydk`**（可匯入 EDOPro 等模擬器）或**複製純文字**卡表。

## 使用方式

### 方法一：直接開啟
用瀏覽器直接開啟 `index.html` 即可（建議 Chrome / Edge / Safari 新版）。

### 方法二：本機起一個簡易伺服器（若直接開啟遇到瀏覽器限制）
```bash
cd Yu-gi-oh
python3 -m http.server 8000
# 然後瀏覽器開 http://localhost:8000
```

## 資料來源
- **卡片資料（中文）**：[ygocdb.com 百鴿中文卡庫](https://ygocdb.com) —— 提供中文卡名與完整中文效果。
- **卡圖**：[ygoprodeck](https://ygoprodeck.com) 卡圖 CDN。

卡片版權屬 KONAMI，本工具僅供學習與卡組規劃之用。

## 專案結構
```
index.html
assets/
  css/style.css
  js/
    data.js      # 主題／流行構築骨架／生成器引擎與泛用卡池（可自行增修）
    api.js       # ygocdb 查詢 + ygoprodeck 卡圖 + 本機快取
    deck.js      # 我的卡組狀態、合法性檢查、.ydk / 文字匯出
    builder.js   # 卡組生成器規則引擎
    ui.js        # 介面：分頁、搜索、彈窗、流行構築、生成器、卡組抽屜
    app.js       # 進入點
```

## 自訂與擴充
想新增流行構築或調整比例，直接編輯 `assets/js/data.js`：
- 卡片以 `{ id, n, q }` 表示（`id` = 卡密＝卡圖檔名，`n` = 中文名，`q` = 張數）。
- 卡密可在「卡片搜索」點開任一卡片、或於 [ygocdb.com](https://ygocdb.com) 查得。
