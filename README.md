# 遊戲王中文卡表工具

一個純前端、開箱即用的 **遊戲王（Yu-Gi-Oh! OCG/TCG）中文卡片工具**。
提供中文卡查、現行環境流行構築、以及依你的條件自動生成卡組，並附卡圖方便辨識。

> 🈶 **全繁體介面**：卡名與效果顯示自動轉為繁體（台灣用字，OpenCC 字表離線轉換）；
> 搜尋可直接輸入繁體，系統會轉回簡體查詢卡庫，保證命中。
> ⚠️ 卡片資料為即時查詢，卡圖亦即時載入，因此**使用時需要能連上網路**。

## 功能

### 🔍 中文卡片搜索
- 直接以**中文或日文卡名**、效果關鍵字搜尋（例：`灰流丽`、`篝火`、`增殖`、`三战之才`）。
- 依卡片種類（怪獸／魔法／陷阱）快速篩選。
- **進階篩選**：依屬性、種族、等級／階級、攻擊力／守備力下限縮小結果。
- 點任一卡片可看**完整中文效果**、攻守、屬性與日/英文名。
- 每張卡右上角「＋」可直接加入「我的卡組」。

### 🏆 現環境流行構築
- 收錄現行環境常見主題（雷火沸動機 Ryzeal、蛇眼、天盃龍、于貝爾、白森林、炎王、
  百夫長 Centur-Ion、拉比林斯 Labrynth、烙印 Branded…），
  附**中文戰術說明**、**構築骨架比例**與**關鍵卡卡圖**。
- 每個主題可「**載入系列全卡**」——即時從資料庫抓取該系列**所有卡片**（永遠是最新收錄）。
- 一鍵「**匯入到我的卡組**」，把整份範例構築帶進卡組編輯器。

> 流行構築為代表性**骨架範例**（依 2025 下半 ~ 2026 上半環境知識整理），比例僅供參考；
> 實際比賽請務必**對照最新禁限表**。

### 🛠️ 卡組生成器（輸入關鍵字，生成可對戰卡組）
**輸入任何主題或卡名關鍵字**（不限現環境，經典主題也行），系統會即時搜尋相關／可連動的卡片，
自動分類（主怪／魔法／陷阱／額外）、依等級決定張數，並配上泛用手坑與萬用額外卡，組出一副能對戰的卡組。
- 例：`青眼`、`剑斗兽`、`元素英雄`、`转生炎兽`、`六武众`、`活死人的呼声`…
- **系列模式**：關鍵字對應到明確系列時，以該系列為引擎聚焦成型。
- **Goodstuff 模式**：單卡或概念關鍵字時，以相關卡＋泛用卡組成骨架（會標示提示）。
- 內建**熱門主題快選**晶片（現環境＋經典），點一下即帶入關鍵字。
- 可調整：卡組風格（連招／控制／快攻）、預算（不限／中等／省錢）、主卡組張數、手坑數、破壞卡數。
- **依卡組性質自動調整比例**：風格選「自動」時會依主題卡組成判斷（陷阱多→控制、低星怪多→連招）；
  選「控制」時會自動補入泛用強力陷阱（神之宣告、反射鏡力、次元障壁…）把卡組填成陷阱牌組。
- **依主屬性配對額外卡**：偵測卡組主屬性後，自動放入對應屬性的靈使（Charmer）連接怪。
- **自動略過無效果卡**：生成時會濾掉無效果的「通常怪獸」，只保留有作用的核心卡（有效果主怪太少時才保留）。
- **每次生成都不同**：加入隨機變化（張數、手坑選擇、順序），配合下方勝率模擬挑出較強的一版。

#### 🎲 生成後自動「模擬對戰」挑選卡組
生成器會對每一版卡組做**起手牌蒙地卡羅模擬**（模擬約 2000 局先手/後手起手），計算：
- **可展開起手率**、**卡手率**、**平均手坑妨害數**、**引擎強度分**，

再對照「主流卡組基準」估算**對局勝率**；並**反覆生成多版、自動挑出勝率最高者**，
盡量達到 **≥60%** 才呈現（達標會顯示 ✅，若多版嘗試仍未達標則標示 ⚠️ 並給出最佳者）。

> 誠實說明：這是**基於起手模擬＋引擎/妨害評分的估計勝率**，用於**相對比較**卡組強度，
> **並非**逐卡結算的完整對局模擬——遊戲王上萬張卡的效果互動無法在純前端完整實作。
> 它能有效區分「能穩定展開的成型卡組」與「卡手／無展開的弱組合」，但不等同實戰勝率。

生成結果會顯示偵測到的主題卡數量與主屬性，並可一鍵「匯入到我的卡組」微調。

> 說明：本生成器以資料庫的即時搜尋為基礎，涵蓋面遠大於少數熱門主題；
> 但它產出的是**合理的骨架**，最佳連招線與張數比例仍建議依你的玩法微調。

### 🗂️ 我的卡組
- 跨分頁共用的工作卡組，資料存於瀏覽器本機（localStorage），關掉頁面也不會消失。
- 自動檢查合法性（主卡組 40–60、額外/副 ≤15、每種卡 ≤3）。
- 一鍵**匯出 `.ydk`**（可匯入 EDOPro 等模擬器）或**複製純文字**卡表。

## 使用方式

### 方法一：直接開啟
用瀏覽器直接開啟 `index.html` 即可（建議 Chrome / Edge / Safari 新版）。

### 方法二：本機起一個簡易伺服器（若直接開啟遇到瀏覽器限制）
```bash
cd Yu-gi-oh
python3 -m http.server 8000
# 然後瀏覽器開 http://localhost:8000
```

## 🌐 部署成線上網站（GitHub Pages）

本工具是純靜態網站，用 GitHub Pages 免費上線。**首次啟用 Pages 需要 repo 擁有者手動點一次設定**（GitHub 的權限限制，無法由程式代開）。有兩種方式，推薦「方式 A」最簡單：

### 方式 A：從分支部署（最簡單，不需工作流）
1. 開 repo 的 **Settings → Pages**。
2. **Build and deployment → Source** 選 **Deploy from a branch**。
3. **Branch** 選你要上線的分支（例：`main`，或本功能分支 `claude/yugioh-card-search-tool-a8jqet`），資料夾選 **`/ (root)`**，按 **Save**。
4. 等 1～2 分鐘，頁面上方會出現網址：`https://<你的帳號>.github.io/Yu-gi-oh/` —— 打開即上線。

### 方式 B：GitHub Actions 部署（用內建工作流）
1. 開 **Settings → Pages**，**Source** 選 **GitHub Actions**。
2. 已合併到 `main` 後會自動部署；或到 **Actions → Deploy to GitHub Pages → Run workflow** 手動觸發。
3. 完成後同樣得到 `https://<你的帳號>.github.io/Yu-gi-oh/`。

> 註：在 Pages 尚未啟用前，Actions 的部署工作流會失敗（這是正常的，啟用後即會成功）。
> 純靜態網站用「方式 A」通常最省事。

## 📱 安裝成 iPhone / iPad App（PWA）

網站上線後，它同時是 **PWA（漸進式網頁應用）**，可加到 iPhone 主畫面像原生 App 使用
（獨立圖示、全螢幕、離線可開介面）：
1. 用 **iPhone 的 Safari** 打開上面的網址。
2. 點下方**分享**按鈕 → **加入主畫面（Add to Home Screen）** → 完成。
3. 主畫面出現「遊戲王卡表」App 圖示，點開即全螢幕運行。

> 說明：iOS 只能透過 Safari 的「加入主畫面」安裝 PWA。
> 若要上架 App Store 的原生 App，需要 macOS + Xcode + Apple 開發者帳號另行封裝
>（可用 Capacitor 等工具包裝本網站），本 repo 未包含該流程。
> PWA 需要網路載入卡片資料與卡圖；App 介面本身可離線開啟。

## 資料來源
- **卡片資料（中文）**：[ygocdb.com 百鴿中文卡庫](https://ygocdb.com) —— 提供中文卡名與完整中文效果。
- **卡圖**：[ygoprodeck](https://ygoprodeck.com) 卡圖 CDN。

卡片版權屬 KONAMI，本工具僅供學習與卡組規劃之用。

## 專案結構
```
index.html
assets/
  css/style.css
  js/
    data.js      # 主題／流行構築骨架／生成器引擎與泛用卡池（可自行增修）
    api.js       # ygocdb 查詢 + ygoprodeck 卡圖 + 本機快取
    deck.js      # 我的卡組狀態、合法性檢查、.ydk / 文字匯出
    builder.js   # 卡組生成器規則引擎
    ui.js        # 介面：分頁、搜索、彈窗、流行構築、生成器、卡組抽屜
    app.js       # 進入點
```

## 自訂與擴充
想新增流行構築或調整比例，直接編輯 `assets/js/data.js`：
- 卡片以 `{ id, n, q }` 表示（`id` = 卡密＝卡圖檔名，`n` = 中文名，`q` = 張數）。
- 卡密可在「卡片搜索」點開任一卡片、或於 [ygocdb.com](https://ygocdb.com) 查得。
